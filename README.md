# Sacred Waters Ministry — Website

Static website for Sacred Waters Ministry, hosted on GitHub Pages.

- `index.html` — the single-page site (home, about, services, leadership, give, contact)
- `assets/logo.png` — ministry seal
- `.nojekyll` — tells GitHub Pages to serve files as-is (no Jekyll processing)

## Hosting
Published with **GitHub Pages** (Settings → Pages → Deploy from branch → `main` / root).
Custom domain `sacredwatersministry.org` is set via the `CNAME` file + DNS at IONOS.

## Editing
Edit `index.html` and commit. Content is plain HTML — service times, leadership,
contact info, and the giving link live directly in the markup.
